### 关注河湖接口

|作者|修改日期|URL| method |
|----|----|----|----|
|李跃龙|2017-12-25|待定|POST|

------------

####请求参数
|参数名|类型| 描述 |是否必填|备注|
| ---------|--- | -------|----- | ---|
|id| Number| 关注的河道ID |是|无|
|follow| Number| 是否关注 |是|1：关注，0：取消关注|


-------------------------------
####返回数据
|参数名|类型| 描述 |备注|
| ---------|--- | -----|------|
|state| Number| 请求状态 | 200：请求成功 |
|data | Object | 数据主体 | 无 |
|data.message | String | 爆料结果提示信息| 无 |

-------------------------------------
#### 返回实例
```javascript
{
    "state": 200,
    "data": {
        "message": "爆料成功！"
    }
}
```
