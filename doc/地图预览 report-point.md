### 地图预览接口

|作者|修改日期|URL| method |
|----|----|---|
|李跃龙|2017-12-21|待定|POST|

------------

####请求参数
|参数名|类型| 描述 |是否必填|
| ---------|--- | ------------ | ---|
|location| String| 坐标经纬度 |是|
|range| Number| 范围|是|

-------------------------------
####返回数据
|参数名|类型| 描述 |备注|
| ---------|--- | -----|------|
|state| Number| 请求状态 | 200：请求成功 |
|data | Array | 数据主体 | 无 |
|data[].location | String | 坐标经纬度 | 无 |
|data[].id | Number | 曝光信息ID | 无 |

-------------------------------------
#### 返回实例
```javascript
{
    "state": 200,
    "data": [
        {
            "location": "112.865569,23.951898",
            "id": "100014"
        }
    ]
}
```
